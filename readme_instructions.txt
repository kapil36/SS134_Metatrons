The folder 4 rfid is meant for running the code on Arduino IDE.
Extract the files and run the Arduino File firmware.

Code Requirements:
Arduino IDE installed on your system.
User needs to have Arduino Uno, RFID Reader(MFRC522) and their tags.

Compile, Upload and then Choose Serial Monitor Option to run the code.

Now scan the tags through multi (in this case only 4 rfid readers)readers.

All the tag id can be seen.

Tags denote the Inmates and Access Readers will be installed inside the rooms.
